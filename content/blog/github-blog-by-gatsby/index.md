## 필수 조건
+ npm
+ node.js


## Gatsby 설치
```
npm install -g gatsby-cli
```

## Gatsby 샘플
https://github.com/gatsbyjs/gatsby-starter-blog

```
gatsby new my-blog-starter https://github.com/gatsbyjs/gatsby-starter-blog
```

## 실행
```
gatsby develop
```
